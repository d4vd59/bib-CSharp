# bib-CSharp
Das ist Bildung innovativ Bereitstellen - bib (C#)

Dies ist die native Desktop-Version der Quiz-App, entwickelt in C#.
Die App bietet dieselben Kernfunktionen wie die mobilen und Web-Versionen, optimiert für PC-Nutzung.

Features

1. Quiz spielen auf dem Desktop

2. Echtzeit-Duelle über Backend

3. Benutzerprofile & Ranglisten

4. Native Windows-Oberfläche (z. B. WPF / WinUI)

Setup

1. Visual Studio installieren (mit .NET Desktop Development workload)

2. Repository klonen: git clone repo-url

3. Projekt öffnen und wiederherstellen

4. Anwendung starten.
